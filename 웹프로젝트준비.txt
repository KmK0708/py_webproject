🎯 프로젝트 주제

“해외 코인 시세 분석 및 시각화 웹 대시보드”
→ Binance API + 웹크롤링을 통해 실시간 시세·거래량 + 뉴스 트렌드를 함께 보여주는 웹 대시보드 제작

💡 프로젝트 개요

목표:
해외 코인 거래소(Binance API)에서 실시간 시세 데이터를 수집하고,
추가로 코인 관련 뉴스 정보를 웹크롤링을 통해 가져와
시세 변화와 뉴스 흐름을 함께 분석·시각화하는 대시보드를 구현한다.

핵심 포인트:

실시간 데이터 수집 (Binance API)

뉴스/이슈 데이터 수집 (웹크롤링)

통계 및 추세 분석

인터랙티브 시각화 (차트/그래프/필터 기능)

데이터베이스 연동 (SQLite/MySQL)

📊 기능 구성
구분	주요 내용
데이터 수집(API)	Binance API를 이용해 주요 코인(BTC, ETH 등) 시세, 거래량, 변동률 수집
데이터 수집(웹크롤링)	CoinDesk, CryptoNews 등 해외 뉴스 사이트에서 코인 관련 기사 제목/시간 크롤링
데이터 저장	시세 및 뉴스 데이터를 주기적으로 DB(MySQL)에 저장
데이터 분석	변동률, 거래량 추이, 뉴스 발생 시점과 시세 변동 비교 분석
데이터 시각화	Plotly·Chart.js를 활용해 실시간 차트 및 뉴스 타임라인 시각화
웹 UI	Flask + HTML/CSS + JS 기반의 대시보드 형태 구성
보안 및 요청 제어	API 및 크롤링 요청 주기 제한(time.sleep), 예외처리, 안전한 요청 처리
⚙️ 사용 기술 스택
분야	사용 기술	설명
언어	Python, JavaScript	백엔드(수집/분석) + 프론트엔드(UI)
백엔드 프레임워크	Flask	서버, API, 라우팅 담당
데이터베이스	SQLite 또는 MySQL	시세·뉴스 데이터 저장
데이터 수집(API)	Binance REST API, requests	실시간 코인 시세 수집
데이터 수집(크롤링)	BeautifulSoup, requests, time.sleep	뉴스/이슈 크롤링 (HTML 파싱)
데이터 분석	pandas, numpy	변동률, 이동평균, 통계 분석
시각화	Plotly.js, Chart.js	가격 추세, 거래량 그래프, 뉴스 타임라인
웹 UI	HTML, CSS, Bootstrap, JS	반응형 대시보드 화면 구성
배포(선택)	Render / Vercel	웹사이트 배포
버전 관리	GitHub	코드 백업 및 포트폴리오 공개용
💾 시스템 구조도 (System Architecture)
[Binance API]         [Crypto News Web Pages]
     ↓ (requests)               ↓ (BeautifulSoup)
───────────────[Data Collector (Python)]───────────────
                     ↓
                 [SQLite / MySQL]
                     ↓
          [Flask Web Server & API]
                     ↓
       [Dashboard UI (HTML + Plotly.js)]


📌 설명 요약

Binance API → 실시간 시세 데이터 수집

Web Crawling → 뉴스 및 이슈 데이터 수집

Flask → DB에서 시세/뉴스 조회 후 시각화 페이지 렌더링

Plotly.js → 시세 그래프 + 뉴스 타임라인 표시

📅 개발 일정 (6~8주 계획)
주차	진행 내용

📍결론 요약

본 프로젝트는 Binance API로 실시간 코인 데이터를 수집하고,
웹크롤링을 통해 뉴스 데이터를 함께 분석하여
“가격 변동 + 뉴스 트렌드”를 통합 시각화하는 대시보드 웹 애플리케이션이다.

데이터 수집 → 분석 → 시각화 → 웹 구현의 전체 흐름을 직접 경험할 수 있는 프로젝트이다.
