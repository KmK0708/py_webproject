🎯 프로젝트 주제

“해외 코인 시세 분석 및 시각화 웹 대시보드”
→ Binance API + 웹크롤링을 통해 실시간 시세·거래량 + 뉴스 트렌드를 함께 보여주는 웹 대시보드 제작

💡 프로젝트 개요

목표:

해외 코인 거래소(Binance)에서 실시간 시세 데이터를 수집

코인 관련 뉴스 정보를 웹크롤링으로 수집

시세 변화와 뉴스 흐름을 함께 분석하고 시각화 대시보드 구현

핵심 포인트:

✅ 실시간 데이터 수집 (Binance API)

✅ 뉴스/이슈 데이터 수집 (웹크롤링)

✅ 통계 및 추세 분석

✅ 인터랙티브 시각화 (차트/그래프/필터)

✅ 데이터베이스 연동 (SQLite/MySQL)

📊 기능 구성
구분	주요 내용
데이터 수집 (API)	Binance API를 이용해 주요 코인(BTC, ETH 등)의 시세, 거래량, 변동률 수집
데이터 수집 (웹크롤링)	CoinDesk, CryptoNews 등 해외 뉴스 사이트에서 코인 관련 기사 제목/시간/링크 크롤링
데이터 저장	시세 및 뉴스 데이터를 주기적으로 DB(MySQL)에 저장
데이터 분석	변동률, 거래량 추이, 뉴스 발생 시점과 시세 변동의 상관관계 분석
데이터 시각화	Plotly.js / Chart.js를 이용한 실시간 그래프 및 뉴스 타임라인 시각화
웹 UI	Flask + HTML/CSS/JS 기반 대시보드 구성
보안 및 요청 제어	API 요청 제한(rate limiting), 예외처리, 크롤링 차단 방지(User-Agent 설정)
⚙️ 사용 기술 스택
분야	사용 기술	설명
언어	Python, JavaScript	백엔드(데이터 수집/분석) + 프론트엔드(UI)
백엔드 프레임워크	Flask	서버, API, 라우팅, 템플릿 렌더링 담당
데이터베이스	SQLite / MySQL	시세·뉴스 데이터 저장 및 관리
데이터 수집 (API)	Binance REST API, requests	실시간 코인 시세 및 거래량 데이터 수집
데이터 수집 (크롤링)	BeautifulSoup, requests, time.sleep	뉴스/이슈 HTML 파싱 및 요청 간격 조절
데이터 분석	pandas, numpy	이동평균, 변동률, 표준편차 등 통계 분석
시각화	Plotly.js, Chart.js	가격 추세, 거래량 그래프, 뉴스 타임라인 시각화
웹 UI	HTML, CSS, Bootstrap, JS	반응형 대시보드 화면 구성
배포(선택)	Render / Vercel	웹사이트 배포 환경
버전 관리	GitHub	코드 관리 및 포트폴리오 공개용
💾 시스템 구조도 (System Architecture)
[Binance API]         [Crypto News Web Pages]
     ↓ (requests)               ↓ (BeautifulSoup)
───────────────[Data Collector (Python)]───────────────
                     ↓
                 [SQLite / MySQL]
                     ↓
          [Flask Web Server & API]
                     ↓
       [Dashboard UI (HTML + Plotly.js)]


요약:

Binance API → 실시간 시세 데이터 수집

Web Crawling → 뉴스 및 트렌드 데이터 수집

Flask → DB 데이터 조회 및 웹 렌더링

Plotly.js → 시세 + 뉴스 타임라인 시각화

📍결론 요약

이 프로젝트는 Binance API로 실시간 시세 데이터를 수집하고
웹크롤링으로 뉴스 데이터를 병합하여
가격 변동과 뉴스 트렌드를 통합 시각화하는 웹 대시보드입니다.

데이터 수집 → 분석 → 시각화 → 웹 구현의 전체 파이프라인을 직접 경험할 수 있으며,
AI·데이터·웹개발 역량을 종합적으로 보여줄 수 있는 프로젝트입니다.
